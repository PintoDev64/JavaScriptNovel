Definición de todas las características y herramientas que contendrá el programa.
### 🧱 Características Base
- **Compilador/Interprete y Librería de Lenguaje Propio:**
	- [ ] **Sintaxis y Nombre** Desarrollar sintaxis y nombre del lenguaje .
		- [✏ Sintaxis](✏%20Sintaxis.md)
	- [ ] **Parser de Lenguaje**: Desarrollar un parser para el lenguaje.
	- [ ] **Transformación a JavaScript**: Convierte las instrucciones del lenguaje personalizado en JavaScript.
	- **Soporte para JavaScript**: Permitir el uso de JavaScript puro en el proyecto para flexibilidad con:
		- [ ] **Gestión de Escenas**: Herramientas para crear, organizar, y manejar escenas de la novela visual.
		- [ ] **Soporte para Imágenes y Sonidos**: Capacidad de manejar imágenes, animaciones, y sonidos.
		- [ ] **Sistema de Diálogos**: Una funcionalidad clave para manejar diálogos entre personajes.
		- [ ] **Manejo de Variables y Estados**: Facilitar la gestión de variables globales, como la puntuación o las decisiones del jugador.
		- [ ] **Soporte para Guardar/Cargar**: Funcionalidad para guardar y cargar el progreso del juego.
